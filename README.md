# 🎨 SkillFlow Embeddable Widget

Beautiful, embeddable skill cards for any webpage, blog, or documentation. Each card is a mini-billboard that drives traffic to [SkillFlow](https://skillflow.builders).

## Quick Start

```html
<!-- Add this where you want the card -->
<div class="skillflow-card" data-skill="credit-optimizer" data-theme="light"></div>

<!-- Add this before </body> -->
<script src="https://cdn.jsdelivr.net/gh/rafsilva85/skillflow-widget@main/skillflow-embed.js"></script>
```

## Features

- **Zero dependencies** — Pure vanilla JS, no frameworks needed
- **Light & Dark themes** — Matches any website design
- **Responsive** — Looks great on mobile and desktop
- **Lightweight** — Under 5KB gzipped
- **Auto-rendering** — Just add the div and script, cards render automatically

## Options

| Attribute | Values | Default | Description |
|-----------|--------|---------|-------------|
| `data-skill` | Skill slug | `demo` | Which skill to display |
| `data-theme` | `light`, `dark` | `light` | Card color theme |
