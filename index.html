🎨 SkillFlow Embeddable Widget

Drop a beautiful skill card into any webpage, blog, or documentation. Each card is a mini-billboard for SkillFlow.

Light Theme

Dark Theme

Usage

<!-- Add this where you want the card -->
<div class="skillflow-card" data-skill="credit-optimizer" data-theme="light"></div>

<!-- Add this before </body> -->
<script src="https://cdn.jsdelivr.net/gh/rafsilva85/skillflow-widget@main/skillflow-embed.js"></script>
